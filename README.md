# Пакет скритов и файлов настроек для ЧПУ станка IMES 4820 

демон IMESd запускается в автостарте, вешается на 1234 порт (константа прописана
в исходнике), принимает соединение, проверяет что идет обращение с разрешенных 
узлов, и пишет все что принял во вновь открытый файл F.nc (в ЧПУ-программе 
открывать всегда его).

клинт IMESc получает в командной строке имя файла, цепляется на прописанный в 
исходнике или явно указанный вторым параметром IP стойки и отсылает ЧПУ 
программу.

В комплект включены файлы настроек для SolidCAM (lic. SSAU)
