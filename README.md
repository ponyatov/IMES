(by) Dmitry Ponyatov <dponyatov@gmail.com>
### Пакет скриптов и файлов настроек для ЧПУ станка IMES 4820 

Для удаленной загрузки по сети реализована схема клиент/сервер
из двух частей (написано на Python 2.7):

Демон IMESd запускается в автостарте на станке, вешается на 1234 порт 
(константа прописана в исходнике), принимает соединение,
проверяет что идет обращение с разрешенных узлов, 
и пишет все что принял во вновь открытый файл F.nc 
(в ЧПУ-программе открывать всегда его).

Клиент IMESc (на рабочей станции) получает в командной строке имя файла,
цепляется на прописанный в исходнике или явно указанный вторым параметром
IP-адрес стойки и отсылает ЧПУ программу.

В комплект включены файлы настроек и шаблонов для SolidCAM 2013 (lic. SSAU)
